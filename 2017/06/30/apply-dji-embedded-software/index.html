2017应聘之大疆

发表于 2017-06-30 | 分类于 让生活更上一层楼 |

本科就选择了方向嵌入式，一直认为它是一个好方向，所以找工作时就比较关注嵌入式的公司，其中一个就是大疆，无人飞行器控制系统及无人机解决方案研发和生产商。

研究生也听师兄提起他诱人的待遇，也正好一位师兄发了大疆的优招，接着就开始了大疆的应聘之路。

投简历+综测

5月26号，师兄发了大疆的优招通知，第二天就在官网投了自己的简历，等待着大疆的消息。

6月2号收到大疆综合测评的通知，说是3/4号两天完成，刚开始以为是技术笔试，毕竟不敢掉以轻心，所以就和同学商量着一起做，等到3号晚上一起做同学的时才发现是心理测评，题目主要是场景设定：让你选择最合理的；心理测试：你是什么样的人；逻辑推理：就是数学中的真假表达式互推；主要是这些。

因为3号我没状态，所以我是4号上午到实验室做的，每个人的题目都一样，前一天看了同学的题目，还有点印象，对时间进度有一定的好处，不过80多道选择题100分钟足矣，我好像只用了半个多小时。

后来才知综合测评刷掉了一半的人，包括和我一起应聘的两位同学。

这里说下我对综测的看法，本科阶段我也不是很理解综测的意义，后来经过华为海思的高管介绍，再加上自己的一些经历，感觉综测对于公司招聘来说确实很重要。主要是公司想要找到合适的人，这样的人不仅技术好，而且性格符合公司的文化，比如大疆这样的公司可能更需要点敢冲敢闯的人，表现可以适当的激进点，毕竟公司还算处于创业期，而对于华为这样的成熟的大公司可能更需要稳重踏实的人，毕竟公司的制度健全，发展稳定，只要不出意外业绩就可以稳步增长，所以表现可以适当的低调含蓄点。当然这是综测的大方向，具体到面试当中就是因面试官而异了。

另外综测不单单是测试性格吻合度，还有心理因素与知识广度与深度，还有逻辑推理能力。
比如测试心理因素的题：一个人说了他的悲惨遭遇，你的表现是怎样的？A. 说出你同样的遭遇；B. 认同他的遭遇并适时规劝；…..显然这一题是考心理知识的：如何规劝心情落入低谷的人。

比如测试知识广度与深度的题：下面哪个公司最牛逼？A. 华为2017年业绩xx，员工xx，年营业额xx；B. 高盛2017年业绩xx，员工xx，年营业额xx；C. 小米一路闯荡到当前的成绩，令人钦佩；…..显然这一题是考知识广度与深度的：哪个公司掌握着当今世界前进的方向盘。

逻辑推理就太多了，就不举例子了。

说这些是为了说明：不要以为综测就没有标准答案，在我眼里，上面三类题目都是有标准答案的，有些看起来是让你选择你的行为，如果你了解这方面的知识完全可以不按照自己做的选择。

好吧，综测就说这么多吧。

假笔试+两次电面

6月9号在参加华为的面试期间收到了大疆的笔试通知。
回到学校首先查了下大疆往年的笔试题目，坑爹的是只能找到2014的嵌入式笔试试卷，一同学报的算法岗，也只能找到2015的试卷，一度怀疑公司估计泄漏的。看了下嵌入式的题目是不简单，主要是c语言，操作系统，cache-mmu，通信协议，简单算法，接下来就是编程设计了，做起来感觉都有点难度，不过都硬着头皮做了一遍，有很多文字描述题，知道怎么一回事，但很难面试完全，不过，知道就行，先不管了。

之前看的现代操作系统，对os有了基本的了解，接下来就是C语言三剑客，不过因为时间紧张，我只看了C陷阱与缺陷，毕竟内容比较少，就这样还打乱了我的C++计划，看完就差不多到时间了。
坑爹的是大疆接下来发了另外一个通知：

你好，在综合考虑多方因素（考试周临近，英语4、6级考试封场，异地同学往返奔波等）后，
我们决定取消原定于本周五的现场笔试，初面改为电话/视频面试，初面时间为6月15日至6月21日，
请同学们在这段时间内注意接听电话，如错过请耐心等待，我们会多次拨打。
对于造成的不便我们深表歉意，后续通知请关注短信和邮件，望同学们理解配合。

大疆创新 招聘中心

坑了我七八十块钱（两个人退票两次）

既然这样那就算了，面试基本都好说，等着吧。

6月16号傍晚，我正在吃饭，广东深圳打来一个电话，一看就知道是大疆的，之前感觉他应该会约个时间来面试，不能突然来个电话直接就开始各种问吧，事实表明我是对的，这点大疆做的还不错，和我约定第二天下午2:40进行电话面试。

6月17号中午，我早早的去吃饭，然后回宿舍进行午休，因为实验室没有一个自由发挥的环境，所以选择在宿舍进行电面，而宿舍电话信号又不是非常好，所以我在靠近阳台的地方搭建了一个简单的桌子，扑上我的简历，一杯水，一张白纸和一支铅笔，这是我的习惯，谈论技术时有支笔可以讲的更有逻辑一点。

午休醒来2点左右，就坐在自制的桌子旁看自己的简历，慢慢等待着时间的流逝。

2点39分，电话来袭，一个年轻人的声音传来。

先做个自我介绍，他感觉不够具体，让我讲了一下我的项目经历，然后我就把我简历中所有的项目从头到尾简单说了一遍。然后他就开始问我工作当中做的一个简单FAT文件系统，已经两三年了，基本都忘记了，我就大概讲了下FAT是怎么一回事，感觉回答的很不好。

接着他看到我最近的机器人项目，问了我音视频通信，我说我们是做面向市场的成熟产品，音视频我们只是用其他商家成熟的方案，对里面的细节不是很了解，然后又问了个多处理器的调度方法，我说我对多处理器不是很了解，调度有个是群调度吧，他又让我解释什么叫群调度，我说将同一工作集的调度单元同时分布在多个处理器上的调度方法，他也没有回应，感觉他不是很满意，很失落，期间我一直陪笑脸，而这面试官一直很严肃，坑爹。

这个面试过程半个小时多一点，心想肯定完蛋，问的几个主要问题都回答的很不好，嗨，既然这样了，不管他了。

第二天和小伙伴交流，他说他有两次电面，因为需要两个面试官在进行电面，而他的面试官的其中一个出差，所以也就变相的有两次电话面试。而在当天下午3：50分，我又收到上海的一个电话，接听了才知道还需要上海的一位面试官来面一次，我们约定半个小时后开始，因为我需要回到宿舍才能行。

一切和第一次电面准备的一样，这次聊的比较开，面试官也特别的和蔼，上来问了我昨天面试感觉如何，我说问的几个问题都回答的不太好，他说问了几个回答都是这样的，然后主要从机器人项目开始，上至网络传输协议，下至底层驱动，中间操作系统，进程调度，虚拟地址和物理地址的转换等等一大堆。

这次还不错，讲了我的学习工作的思想，他说和大疆的理念还是比较相符的。

因为有了第二次面试感觉好了很多，接下来就等着结果了。

假真笔试+终面

6月21号下午7点半左右收到大疆的通知，说让本周六(6月24日)去参加笔试和终面，详情会有邮件通知；

6月23日，我和同学订了去南京的票，就近订了宾馆，吃过晚饭就开始在宾馆准备相应的内容，我的操作系统虽然感觉不太深入，但感觉还行，所以就快速浏览了下《超标量乱序处理器设计》一书中的tlb和cache部分。

6月24日，八点起床洗漱下，吃个早餐，8点50的样子，到南京总统府亚朵酒店，直接到达六楼，进入一个可以容纳八九十人的会议室，我们走进去，大疆的老师（就叫这个吧）每人发了一个无人机的纸模（可以拆开拼出无人机），由于人基本都坐满了，所以我们就坐在了窗户旁边的板凳上。陆续来人后，我数了下，总共80人的样子。

总共两套卷子，半个小时一套：一套逻辑推理题，一套数字逻辑题。

本来以为是技术题目，表示懵逼。

第一套试卷因为还没进入状态，而且看同学也不紧不慢的做，自己也就懈怠了，半个小时过去了，估计还没做一半，其他都是胡乱写的答案，坑，逻辑推理很绕人，开始感觉还能做，后来都蒙了，真个是懵逼。

第二套试卷就还好了，纯数字逻辑，有了第一套的经验，第二套我们分开做，我从前往后，同学从后往前，半个小时过去了，我做了一大半，同学就选择一个选择题，更懵逼，一脸黑线，其他的蒙吧…….–..–

两套试卷之后就开始了面试环节，我和同学的是下午一点，所以就直接回宾馆睡觉去了。

由于宾馆12点退房，我们推迟到12点半，然后吃个饭就去亚朵酒店，准备在那休息下，刚坐下大疆的老师就说：下午2点之前面试的嵌入式的，结构的…过来登记下，然后去12楼等待，靠，这就直接去面试了啊，还差一个小时呢。

嵌入式的我前面还有两位，一起到面试嵌入式的房间门外，里面有一位正在面试，我们三个就聊了一会，他们两个男生是东大信息的，自我感觉很牛逼的样子，做FPGA图像算法实现的，女生浙大的，表现特别紧张，做FPGA传感器的。

他们的面试情况了解的不是太多，只知道男生出来一脸黑线，跑到另外一个房间去了，后来才发现去写程序了，说是一个滤波算法，反正我不会，女生出来我接着就被叫进去了，就没了解到。

我进去后坐在椅子上，两位面试官，一个在沙发上坐着，以为在上厕所，我拿出我的两张简历交给了沙发上的面试官，我说先做个自我介绍，他说先等等上厕所的那位同事，他就先看了我的简历，那位同事回来后，我就开始了我的自我介绍，后来的面试官听到我简介中提到写些小工具，就问了下，我说了我写了哪些工具一类的。

接着机器人项目问题：

1. 机器人项目的架构（负责人）：画图分析整个架构；
2. 我做了哪些事：每个部分我做的都说了一遍；

我自我介绍中提到我的了解范围：cpu架构/cache/内存/进程/文件系统，他接着问了这些：

1. 内核空间与用户空间的区别；
2. 内核空间与用户空间如何通信；
3. mmap是将内核空间映射到用户空间还是反过来？
    （我的回答是将用户空间映射到内核空间，举了一个网卡的例子，不知道对不对，他也不给反馈，黑线）
4. 虚拟地址是如何映射到物理地址的（这一部分特别重要，我单开一个博客讲）
5. 信号量与自旋锁的区别；
6. 信号量与互斥锁的区别；
7. 让写了个生产者消费者的程序。

后面又问了我对大疆的了解，我直接说我确实对大疆的了解不多，只说了大疆做飞行器的，全球市场份额在一半以上，感觉答的很不好。

我为什么想加入大疆：师兄推荐，思想哲学温和。

应该四十分钟左右，问的东西挺多的，我一直嘟嘟嘟嘟嘟的讲。

面完感觉回答的都还行，程序是到外面写的，所以从网上搜索了下案例。

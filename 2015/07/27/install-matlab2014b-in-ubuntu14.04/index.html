linux安装matlab记录

By pumpkin

发表于 2015-07-27

文章目录
  1. 1. 把镜像文件中的install.jar文件替换为MATLAB_R2014B_MAC_LINUX_crack中的install.jar
    1. 1.1. 将R2014b_glnxa64.ISO解压到随便某一个目录下（注意：使用ubuntu自带的工具解压，即右键->archive manager），本教程将解压后的放在/home/lab302/matlab2014b中
    2. 1.2. 将破解文件MATLAB_R2014B_MAC_LINUX_crack中的install.jar复制到matlab2014b/java/jar/下的覆盖原来的install.jar，并执行指令chmod +x *
    3. 1.3. 进入到java/jar文件夹下，可以看到有install.jar文件，执行命令chmod +x *。然后将该文件删除rm install.jar
  2. 2. 安装matlab
  3. 3. 输入installation key: 29797-39064-48306-32452，安装路径这里选择/usr/local/MATLAB/R2014b
  4. 4. 不联网激活：添加MATLAB_R2014B_MAC_LINUX_crack下的license.lic文件激活
  5. 5. 把libmwservices.so文件放入到/usr/local/MATLAB/R2014B/bin/glnxa64文件下
  6. 6. 进入到/usr/local/MATLAB/R2014b/bin文件夹中，输入./matlab启动matlab。
  7. 7. 至此安装完成。

安装所采用的镜像为R2014b_glnxa64.ISO。破解文件分有两个，一个为libmwservices.so文件，另一个为MATLAB_R2014B_MAC_LINUX_crack文件夹[详见共享目录：/192.168.0.227/01.专业软件/matlab/ubuntu-matlab2014b]。安装步骤如下：

把镜像文件中的install.jar文件替换为MATLAB_R2014B_MAC_LINUX_crack中的install.jar

将R2014b_glnxa64.ISO解压到随便某一个目录下（注意：使用ubuntu自带的工具解压，即右键->archive manager），本教程将解压后的放在/home/lab302/matlab2014b中

将破解文件MATLAB_R2014B_MAC_LINUX_crack中的install.jar复制到matlab2014b/java/jar/下的覆盖原来的install.jar，并执行指令chmod +x *

进入到java/jar文件夹下，可以看到有install.jar文件，执行命令chmod +x *。然后将该文件删除rm install.jar

安装matlab

回到解压后的文件目录matlab2014b/下，可以看到有intall文件，执行sudo ./install命令，开始安装【注意：选择使用文件安装秘钥】

输入installation key: 29797-39064-48306-32452，安装路径这里选择/usr/local/MATLAB/R2014b

不联网激活：添加MATLAB_R2014B_MAC_LINUX_crack下的license.lic文件激活

把libmwservices.so文件放入到/usr/local/MATLAB/R2014B/bin/glnxa64文件下

进入到/usr/local/MATLAB/R2014b/bin文件夹中，输入./matlab启动matlab。

至此安装完成。
